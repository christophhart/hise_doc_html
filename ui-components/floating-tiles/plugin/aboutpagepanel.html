AboutPagePanel

A Panel that shows custom and autogenerated plugin information

ID Default Value Description
Font Oxygen Bold Select a font
FontSize 14.0 Set the fontsize
ShowProductName true Shows the product name set in the Project Settings
UseCustomImage false Set a image path to display an image on the AboutPagePanel
CopyrightNotice false Show a copyright notice string
ShowLicensedEmail true Select true if you want to show the licensed email adress.
ShowVersion true Set to true if you want to display the current Product Version number, set in the project settings.
BuildDate true Set the Build time
WebsiteURL true Set a link to a website.

The AboutPagePanel diplays project specific information.
