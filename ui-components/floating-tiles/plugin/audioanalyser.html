Audio Analyser

Connects to module type: Analyser

A floating tile that connects to an Analyser Module and visualizes the audio signal

ID Default Value Description
Font Oxygen Bold no description
FontSize 14.0 no description
ProcessorId "" the Analyser module that collects the data for the visualisation
Index -1 The visualisation type.

You can connect the AudioAnalyser Floating Tile to the Analyser Module with the property "ProcessorId". Just put in the Analysers Processor ID to connect them. 

{
  "ProcessorId": "Analyser1",
  "Index": 0
}
