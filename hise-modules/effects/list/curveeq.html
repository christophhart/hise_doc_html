Parametric EQ

Type ID: CurveEq
Interface classes: RoutingMatrix

A Parametric EQ with addable bands and FFT background


Parameters

# ID Description
0 Gain The gain in decibels if supported from the filter type.
1 Freq The frequency in Hz.
2 Q The bandwidth of the filter if supported.
3 Enabled the state of the filter band.
4 Type the filter type of the filter band.
5 BandOffset the offset that can be used to get the desired formula.
