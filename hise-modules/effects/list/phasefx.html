PhaseFX

Type ID: PhaseFX
Interface classes: RoutingMatrix

A general purpose phase modulation effect.


The PhaseFX module can be used to create all kinds of phase modulation effects - from vowel filters over traditional phasers to comb filters.

Parameters

# ID Description
0 Frequency1 the bottom frequency
1 Frequency2 the upper frequency
2 Feedback the amount of feedback
3 Mix the wet amount

Chains

# ID Restriction Description
0 Phase Modulation All types -
