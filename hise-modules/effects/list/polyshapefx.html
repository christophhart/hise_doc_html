Polyshape FX

Type ID: PolyshapeFX

A polyphonic wave shaper.


Parameters

# ID Description
0 Drive How much you want to drive the input signal into the shaper.
1 Mode Select a shaper function.
2 Oversampling Select if you want to oversample the shaper to get rid of artefacts.
3 Bias The bias amount.
