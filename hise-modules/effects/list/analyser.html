Analyser

Type ID: Analyser
Interface classes: RoutingMatrix

An Analyser Module to analyse and visualise generated sounds.


The analyser gives you three forms of visualisations of the incoming audio signal. You can put these on your interface with the Audio Analyser Floating tile.

Goniometer

Oscilloscope

Spectral Analyser

Parameters

# ID Description
0 PreviewType The index of the visualisation type.
1 BufferSize The buffer size of the internal ring buffer.
