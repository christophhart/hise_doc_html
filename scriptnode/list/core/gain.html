Gain

core.gain

The gain module applies a gain change (multiplication) to a signal.

Parameters

ID Range Default Description
Gain -100.00 - 0.00 0.00 the volume in decibels
Smoothing 0.00 - 1000.00 20.00 the smoothing time for gain changes. Uses a linear ramp between changes.
ResetValue -100.00 - 0.00 0.00 this value will be used as start ramp value whenever the node is being resetted (eg. when the network is unbypassed).
