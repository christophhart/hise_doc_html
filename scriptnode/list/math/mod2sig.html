mod2sig

math.mod2sig

Converts a mod signal to an audio signal

Parameters

ID Range Default Description
Value 0.00 - 1.00 0.00 no description.


This reverses the process of the sig2mod node and converts a signal from the [0...1] range back to the [-1 ... 1] range.
