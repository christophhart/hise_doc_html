container.split

A 3 band frequency split setup

Parameters

ID Range Default Description
Band 1 20.00 - 20000.00 68.07 no description.
Band 2 20.00 - 20000.00 1000.00 no description.

This template will create a frequency splitter between two bands. It takes care of setting up a bunch of jlinkwitzriley filters with the correct filter modes and creates parameters in the container that will move the filter frequency of the respective filter nodes to implement a frequency band split.
