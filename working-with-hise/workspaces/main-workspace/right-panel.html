Right Panel

The Right Panel features five preselected HISE Floating Tiles to ease your workflow when working in the Module Panel.

  • In the MacroTable you can adjust and refine the Macro values that you can set with the Macro Controls in the main topbar.
  • The ScriptWatchTable , InterfacePropertyEditor and Console help you out when you use a Script Processor directly in the Module Panel.
  • The Plotter visualizes the modulation values of Modulators when you hit the visualization button in each modulators topbar.
