Sampler Workspace

With the Sample Editor you get a dedicated workspace for mapping samples and playing them back in a Sampler .

It shows:

  • a Sample Editor to edit single samples.
  • a Map Editor to map the samples into SampleMaps .
  • a Sample Table which lists all samples loaded into the current Sample Map.

If you want to change the Settings of the sampler you can do so by opening the Sampler popup in the Module Tree .
