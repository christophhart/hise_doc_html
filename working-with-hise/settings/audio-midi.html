Audio Settings

Audio

Driver

The driver type. On Windows you should choose ASIO (on macOS it's CoreAudio by default)

Device

The sound card that is used to output the sound

Output

The output channel if your audio interface has multple outputs

Samplerate

The samplerate that HISE uses for rendering the audio. Make sure that this matches the sample rate on your audio interface, or you will get artifacts.

Buffer Size

The buffer size in samples that will be used to render the audio. The best numbers are power-of two numbers, which should be preffered if possible.

MIDI

Midi Input

The MIDI input device list. It should get updated automatically if you connect a new MIDI device - if not try restarting HISE

Midi Channels

A list of all MIDI channels that should be processed. By default all channels are processed, but you might want to filter out some channels depending on your setup.
